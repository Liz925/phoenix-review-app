# Phoenix Review App v1

这是一个移动端优先的个人复盘 PWA：每日复盘、随机想法、阶段复盘、JSON/TXT 导出。

## 最稳使用方式

1. 把整个文件夹部署到 GitHub Pages、Netlify 或任何静态网页空间。
2. 用 iPhone Safari 打开部署后的链接。
3. 选择“添加到主屏幕”，之后就像 App 一样打开。
4. 每天记录，周末导出 TXT 或 JSON 发给 ChatGPT 做分析。

## 注意

- 数据保存在浏览器本地 localStorage，不会上传到服务器。
- 换设备、清缓存、换浏览器前，请先导出 JSON 备份。
- 不建议在 ChatGPT 内置网页预览里使用，部分按钮脚本可能被限制。
